# Privacy Policy for ResumeFit

Last updated: 2026

ResumeFit is a Chrome extension that helps users select the best resume for a job description.

## Data Collection
ResumeFit does not collect, transmit, or store any personal data on external servers.

## Local Data Storage
The extension stores resume versions locally in the user's browser using Chrome's storage API.  
This data remains entirely on the user's device and is never transmitted externally.

## Website Content Access
When the user clicks the "Find My Best Resume" button, the extension reads the job description text from the current webpage to perform a local comparison with saved resumes.

## No Data Sharing
ResumeFit does not:
- send data to external servers
- track users
- collect analytics
- sell or share user information
