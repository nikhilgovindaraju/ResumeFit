# Privacy Policy for ResumeFit

Last updated: March 6, 2026

ResumeFit is a Chrome extension that helps users select the best resume 
for a job description.

## Data Collection
ResumeFit does not collect, transmit, or store any personal data on 
external servers. All processing happens locally on your device.

## Local Data Storage
Resume text that you save is stored locally in your browser using 
Chrome's `storage.local` API. This data never leaves your device and 
is not accessible to any third party, including the developer.

## Website Content Access
When you click "Find My Best Resume", the extension reads the job 
description text from the active tab solely to perform a local 
comparison against your saved resumes. This text is never stored 
or transmitted anywhere.

## Data Deletion
You can delete all stored resume data at any time by:
- Using the Delete button inside the extension, or
- Uninstalling the extension, which removes all locally stored data

## No Data Sharing
ResumeFit does not:
- Send any data to external servers
- Share data with third parties
- Track users or browsing behavior
- Collect analytics or crash reports
- Sell or monetize user information in any way

If this policy changes, the updated version will be posted at this URL
